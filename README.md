vim-plugins
===========

Prerequisite
-----------
cscope (http://cscope.sourceforge.net/)

vim

ctags

The List of Plugins
--------------------
Vundle.vim.git
cscope_maps.vim (http://cscope.sourceforge.net/cscope_maps.vim)

Installation
--------------------
1. [`./install.sh`]
2. Copy the vimrc file to the your vimrc proifile (usually [`~/.vimrc`])
3. Open [`vim`] and type [`:PluginInstall`]
